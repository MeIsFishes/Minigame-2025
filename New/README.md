# 字母射击游戏

一个基于HTML5 Canvas的打飞机游戏，使用键盘字母键发射子弹。

## 功能特点

- ✨ **真实键盘布局**：子弹发射位置完全对应QWERTY键盘的三排布局
  - 第一排：Q W E R T Y U I O P
  - 第二排：A S D F G H J K L（向右偏移）
  - 第三排：Z X C V B N M（向右偏移更多）

- ⏱️ **独立冷却系统**：每个字母键有独立的1秒冷却时间

- 🎮 **UI与游戏同步**：底部键盘UI的每个按键位置精确对应其在游戏画布中的发射位置

- 🎯 **得分系统**：击中敌机获得10分

- 💥 **视觉效果**：
  - 爆炸粒子特效
  - 星空背景
  - 冷却进度显示
  - 按键高亮反馈

- 🔊 **音效**：射击和游戏结束的简单音效

## 如何游玩

1. 在浏览器中打开 `index.html`
2. 点击"开始游戏"
3. 按下A-Z任意字母键从对应键位位置发射子弹
4. 击中横向移动的敌机获得分数
5. 当敌机到达底部时游戏结束

## 技术实现

游戏采用模块化设计，分为以下文件：

- `index.html` - 游戏主页面
- `styles.css` - 样式文件
- `weapon.js` - 武器系统（子弹发射、冷却机制、位置计算）
- `enemy.js` - 敌机系统（生成、移动、碰撞检测）
- `ui.js` - UI系统（键盘显示、冷却可视化）
- `menu.js` - 菜单系统（开始、结束、重新开始）
- `game.js` - 主控制器（游戏循环、渲染、系统整合）

### 位置对应算法

键位在屏幕上的水平位置根据以下规则计算：
1. 每排按其键数均匀分布
2. 第二排向右偏移0.25个键位宽度
3. 第三排向右偏移0.75个键位宽度
4. UI按键使用绝对定位，精确匹配游戏画布中的发射位置

## 浏览器兼容性

需要支持以下特性的现代浏览器：
- HTML5 Canvas
- ES6 Classes
- Web Audio API（可选，用于音效）
